# ANNIE_PicoLaser
PMT timing analysis for the PILAS picosecond diode laser source for the ANNIE experiment.

### Scripts
- `2023_PMT_Timing_Analysis.ipynb` - machinery for performing the offset corrections and the primary peak fitting to determine the instrinsic PMT timing jitter
- `Analyze_Laser_intensity.ipynb` - visualizing the laser intensity and the charge / timing distributions


The laser ntuples used in this analysis can be found here: `/pnfs/annie/persistent/users/doran/datasets/laser_data/`
